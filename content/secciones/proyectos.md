## Proyectos y Repositorios en GitHub

### Herramientas y Librerías GIS
- **gishndev** (https://github.com/lalgonzales/gishndev): Paquete Python para procesamiento de datos geoespaciales, con herramientas para análisis SIG y teledetección. Incluye automatización de flujos de trabajo con Google Earth Engine y QGIS.
- **adef-tools** (https://github.com/lalgonzales/adef-tools): Herramientas para análisis de deforestación, procesamiento de alertas forestales y visualización de datos espaciales.
- **geo-intro-py** (https://github.com/lalgonzales/geo-intro-py): Introducción a Python para análisis geoespacial. Materiales educativos con Jupyter Book, notebooks interactivos y ejercicios prácticos en español.

### Aplicaciones Web y Datos
- **ambsite** (https://github.com/lalgonzales/ambsite): Sitio web para análisis ambiental, con integración de datos geoespaciales y visualizaciones interactivas.
- **lalgonzales.github.io** (https://github.com/lalgonzales/lalgonzales.github.io): Sitio personal con portafolio de proyectos GIS y análisis de datos.

### Educativos y Demostraciones
- **30DaysChallenges** (https://github.com/lalgonzales/30DaysChallenges): Retos diarios de programación y análisis de datos, enfocados en Python y GIS.
- **datasets** (https://github.com/lalgonzales/datasets): Colección de datasets para análisis geoespacial y machine learning.
- **scriptsVarios** (https://github.com/lalgonzales/scriptsVarios): Scripts diversos para automatización de tareas GIS, procesamiento de imágenes satelitales y análisis de datos.

### Otros Proyectos
- **sefer-torah-online** (https://github.com/lalgonzales/sefer-torah-online): Recursos digitales para estudio de la Torá, con contenido multilingüe.
- **FERM-DEMO-PAIS** (https://github.com/lalgonzales/FERM-DEMO-PAIS): Demostración de herramientas para evaluación de recursos forestales.
- **ai-assistance-guides** (https://github.com/lalgonzales/ai-assistance-guides): Guías para asistencia con IA en proyectos de datos y GIS.

### Forks y Contribuciones
- **geemap** (fork de https://github.com/gee-community/geemap): Librería Python para análisis geoespacial con Google Earth Engine.
- **dask-geopandas** (fork de https://github.com/geopandas/dask-geopandas): Paralelización de GeoPandas con Dask.
- **geog-312** (fork de https://github.com/giswqs/geog-312): Introducción a programación GIS.
- **r4ds** (fork de https://github.com/hadley/r4ds): R for Data Science.
- **xlsform** (fork de https://github.com/XLSForm/xlsform): Herramientas para formularios XLS.
